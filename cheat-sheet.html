HTML5

  • <header>

    represents a container for introductory content or set of navigational links

    <header>Your title here</header>
  • <footer>

    defines a footer for a document or section

    <footer>basic information here</footer>
  • <section>

    defines sections in a document such as chapters headers, footers or any other sections of the document

    <section>hello i am s section with a lot of text</section>
  • <article>

    specifies independent self contained content

    <article>I am self contained content</article>
  • <nav>

    it defines a set of navigation links

    <nav>I contain navigation links</nav>
